# Racha — Seguimiento de Hábitos

App gratuita de seguimiento de hábitos. Fase 1: núcleo funcional (auth, onboarding, CRUD de hábitos, check-in diario, PWA).

## Stack

- React 18 + Vite
- Supabase (Auth + Postgres, con RLS)
- PWA (manifest + service worker)
- Deploy: Netlify

## Setup rápido

Ver [SETUP.md](./SETUP.md) para la guía completa (crear proyecto Supabase, variables de entorno, migraciones).

```bash
npm install
npm run dev
```

Abre http://localhost:3000
